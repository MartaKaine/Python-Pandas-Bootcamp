# Data Analysis with Python (Pandas)

This repository contains exercises and analysis completed as part of my **Data Technician Bootcamp (Week 6 – Python & Pandas)**.  
The focus of this project was learning how to use **Pandas for data exploration, cleaning, and analysis**, working with retail and sales datasets in a Jupyter/Google Colab environment.

The exercises simulate typical tasks performed in data roles such as filtering datasets, cleaning missing data, performing aggregations, and visualizing trends.

---

## Skills Demonstrated

During this project I practiced working with **Python’s Pandas library** to manipulate and analyze tabular data.

Key skills include:

- **Loading and inspecting datasets** using Pandas
- **Filtering and subsetting data**
- Using **`.loc[]` and `.iloc[]`** for row and column selection
- **Sorting and organizing data** using `sort_values()`
- **Handling missing values** with `dropna()` and `fillna()`
- **Aggregating data** using `groupby()`
- **Summarizing data** with descriptive statistics
- **Data cleaning and preparation**
- **Basic data visualization** using built-in plotting and libraries like `matplotlib`

These techniques were used to explore patterns in **retail and sales data**, helping identify trends and insights from the dataset.

---

## Data Analysis Tasks

Examples of analysis performed in the notebooks include:

- Filtering datasets to focus on specific **products, regions, or time periods**
- Subsetting columns and rows using **`.loc[]` and `.iloc[]`**
- Sorting datasets to identify **top-performing products or categories**
- Grouping data by categories (e.g., product or region) using **`groupby()`**
- Calculating **totals, averages, and other summary statistics**
- Cleaning datasets by **handling missing values**
- Creating **simple visualizations** to better understand trends in sales data

These tasks reflect the kind of **data preparation and exploratory analysis** commonly required before performing deeper analytics.

## Data Visualization

Basic visualizations were created to help interpret the data, including:

- Bar charts for comparing product or category performance
- Line charts for identifying trends over time
- Quick exploratory plots generated directly from Pandas DataFrames

Libraries used:

- **Pandas built-in plotting**
- **Matplotlib**

Visualization helps translate raw data into **clear insights and patterns**.
